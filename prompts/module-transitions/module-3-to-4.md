# Module 3 -> Module 4: Outline to Drafting

Purpose: convert outline into coherent academic draft paragraphs.

Output: introduction, body paragraphs, counterargument/rebuttal when appropriate, conclusion, signal devices, hedging, and citation placeholders.

Formatting: prose paragraphs separated by `\n\n`; never one glued block.

Quality rule: never draft about the outline itself. Do not output labels or meta-template phrases such as `Introduction plan`, `Topic sentence:`, `Evidence to use:`, `Analysis purpose:`, `Link back:`, `The student should`, or `the strongest body paragraph should`. Use those outline fields only to compose real essay paragraphs.

Citation behavior: use `[citation needed]` for unsupported factual claims.

Failure behavior: draft cautiously from supplied outline only; do not add unsupported facts.

Instruction:
- Convert the outline into Module 4: Drafting.
- Use paragraph form. Include an introduction with background, importance, thesis, and thesis map.
- Each body paragraph should include a topic sentence, evidence, analysis, and link back to thesis.
- Use academic tone: formal, precise, unemotional, balanced.
- Use metadiscourse and signal devices where helpful: firstly, in addition, however, for example, therefore, this essay argues.
- Use hedging for uncertain claims: may, could, appears to, suggests.
- Do not invent citations; use `[citation needed]` for unsupported factual claims.
- Do not write about outline labels such as Introduction plan, Topic sentence, Evidence to use, Analysis purpose, or Link back. Use them only to compose real paragraphs.

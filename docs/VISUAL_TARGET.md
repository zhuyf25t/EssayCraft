# EssayCraft Visual Target

Use these reference images:

- `public/reference-images/mockup-1-dashboard.png`
- `public/reference-images/mockup-2-drafting-workspace.png`

The implementation does not need pixel-perfect fidelity, but should capture:

- clean white/off-white background
- marker/crayon feeling for EssayCraft title
- pastel highlighter text colors
- left module pipeline
- central editor card
- right AI Assistant panel
- version/snapshot panel
- bottom Highlight Key
- large primary module-generation button

Prioritize product utility and stability before decorative detail.
